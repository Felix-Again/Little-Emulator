# Little-Emulator

Little project to emulate a specific vintage console cpu and how it works.

I will aim to make it be able to run assembly code made for this specific vintage console that I will not specify...

Have fun if you want to download this messy code or want to see me find out how to implement everything that I need to. Thanks!

I used the following websites as a theorical base of what to do:
[Rylev Gameboy Emulator Book](https://rylev.github.io/DMG-01/public/book)
[Meganesu OPCodes page](https://meganesu.github.io/generate-gb-opcodes/)
[RGBDS Instructions list and descriptions](https://rgbds.gbdev.io/docs/v1.0.1/gbz80.7)
